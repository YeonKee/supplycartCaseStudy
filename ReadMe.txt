1. Download the project source code and put it inside the htdocs folder under xampp.
2. Open XAMPP, start service for Apache and MySQL.
3. Go to Chrome, enter “localhost/phpmyadmin”.
4. Create a database, enter database name “supplycartcs”, leave the rest default.
5. Import the tables and data, select supplycartcs.sql file given.
6. Open the source code at VS Code
7. Open 2 new terminal in the VS Code, run both commands under different terminal
- npm run dev
- php artisan serve
8. Go to http://127.0.0.1:8000/
